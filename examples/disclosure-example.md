# Disclosure Statement Example

This is a filled example of a standard-form disclosure for a completed portfolio project.
Blank template at `templates/disclosure-statement.md`.

---

## Standard Form Disclosure

> **AI Disclosure:** This portfolio site was produced through structured human-AI collaboration using Claude (claude-sonnet-4-6, Anthropic, 2026). Maya Chen directed all substantive decisions, including the visual identity, project sequencing, and editorial voice. The AI contributed layout structure options during the exploration phase, generated initial HTML and CSS scaffolding for the project grid component, and drafted copy for two project descriptions that were subsequently rewritten. All code was reviewed and tested before deployment. Three Records of Resistance document specific decisions to reject or revise AI suggestions.

---

## What the User Contributed

- Visual identity: palette, typography, grid system (see Records of Resistance #1)
- Project selection and sequence: which projects appear, in what order, and why
- All final copy: "About" section, project titles, project descriptions (AI drafts revised substantially or discarded)
- Layout direction: the final three-column asymmetric grid was the user's concept; AI generated structural alternatives that were not used

## What AI Contributed

- Layout alternatives during the exploration phase (three options generated; none adopted as-is)
- HTML and CSS scaffolding for the project grid (used as a starting point, refactored by the user)
- First-draft copy for two project descriptions (rewritten; AI framing removed)
- Accessibility audit recommendations (adopted without revision)

## Where the User Overrode AI

1. **Color palette:** AI suggested a contemporary sage-and-cream palette. User kept their established black, off-white, and terracotta system. (Record of Resistance #1)
2. **Grid structure:** AI recommended a standard two-column card grid for projects. User chose an asymmetric three-column layout that reflects the spatial quality of their design practice. (Record of Resistance #2)
3. **"About" copy framing:** AI drafted an "About" section that opened with credentials and education. User rewrote it to open with a description of their practice, placing credentials in a secondary position. (Record of Resistance #3)
