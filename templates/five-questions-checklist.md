# Five Questions Checklist

Run this at every review point: before accepting AI output, before submitting work, before publishing.

**Project:** [Project name]
**Date:** [YYYY-MM-DD]
**Review point:** [e.g., "Before submitting draft to editor", "Before merging PR"]

---

| # | Question | Yes/No | Notes |
|---|----------|--------|-------|
| 1 | **Can I defend this?** Could I explain and justify every claim, choice, and structure in this work to a knowledgeable audience? | | |
| 2 | **Is this mine?** Did I direct this work, or did I accept the AI's framing without evaluating it against my own position? | | |
| 3 | **Did I verify?** Have I checked facts, sources, code behavior, or logical consistency independently of what the AI told me? | | |
| 4 | **Would I teach this?** Do I understand this well enough to explain it to someone else and answer their questions? | | |
| 5 | **Is my disclosure honest?** Does my AI Use Log and disclosure statement accurately describe what I did and what AI did? | | |

---

- [ ] All five pass. Proceed.
- [ ] One or more fail. Stop and address before continuing.
